Lokale Datenspeicherung

Diese App speichert alle Daten ausschließlich lokal auf deinem Gerät (localStorage des Browsers). Es werden keine Daten an Server übertragen, keine Cookies gesetzt und keine Analyse-Tools verwendet.

Gespeicherte Daten

Folgende Daten werden lokal gespeichert:

• Checklisten-Status (welche Punkte abgehakt sind)
• Kosteneintragungen (Kraftstoff, Stellplatz, Sonstiges)
• Fahrzeugmaße und Kalibrierungswerte für das Nivelliertool
• App-Einstellungen (Theme, Toleranz)

Löschen der Daten

Die gespeicherten Daten können jederzeit über die Browser-Einstellungen (Websitedaten löschen) oder durch Deinstallation der PWA entfernt werden.

Externe Dienste

Die App lädt keine externen Ressourcen. Alle Schriftarten und Inhalte sind lokal eingebunden. Es werden keine Verbindungen zu Drittanbietern hergestellt.

Gerätesensoren

Das Nivelliertool verwendet den Lagesensor (Gyroskop) des Geräts. Die Sensordaten werden nur lokal zur Berechnung der Neigung verwendet und nicht gespeichert oder übertragen. Auf iOS-Geräten wird die Erlaubnis vorab abgefragt.

Hosting

Die App wird über GitHub Pages bereitgestellt. Beim Aufrufen der Seite können durch GitHub (Microsoft) Server-Logs erstellt werden. Weitere Informationen: GitHub Pages Datenschutz.
